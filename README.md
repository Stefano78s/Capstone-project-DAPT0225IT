Fiber Vision

Questo report analizza la copertura broadband in Italia, con particolare attenzione alla tecnologia FTTH (Fiber to the Home) e FTTC (Fiber to the Cabinet).
L’obiettivo è fornire una panoramica chiara su:
Estensione della copertura territoriale
Famiglie raggiunte
Differenze regionali
Presenza degli operatori sul territorio
Trend evolutivo
Il report è navigabile e permette una lettura semplice tramite grafici, mappe e tabelle.

Struttura del Report
Il report è organizzato in due sezioni principali.

Cablaggio FTTH — Overview nazionale
Include:
% Copertura nazionale FTTH
Famiglie FTTH / Famiglie totali
Andamento temporale della copertura (2023–2025)
Mappa regionale colorata sulla base della copertura
Tabella regionale con:
Celle FTTC / FTTH
Famiglie totali
Famiglie raggiunte FTTH
Punti terminazione FTTH
Obiettivo: comprendere il livello di copertura FTTH a livello nazionale e regionale e come evolve nel tempo.

Presidio Copertura Operatore
Mostra dove operano i singoli operatori e con quale estensione:
Numero di regioni/province/comuni coperti per operatore
Classifica operatori per diffusione
Mappa interattiva dei territori serviti
Tabella dettagliata:
Operatore
Regione
Provincia
Comune
Obiettivo: evidenziare il presidio degli operatori sul territorio e confrontarne l’ampiezza.

Dataset utilizzati
Tabella	Contenuto
Copertura_Regionale	Copertura FTTH/FTTC aggregata per regione
Fiber_Vision_Actual	Dati storici sui comuni
network_operator	Associazione operatori–territori
Regioni / Comuni	Tabelle geografiche
Calendar	Gestione anni

Logica dei calcoli (DAX)
Le misure principali sono calcolate sull’ultimo periodo disponibile per l’anno filtrato.
Tra le principali:
% Copertura Nazionale
Famiglie totali (ultimo riferimento)
Famiglie FTTH (ultimo riferimento)
Celle FTTH / FTTC %
Punti terminazione FTTH
Queste misure garantiscono che l’analisi si basi sempre sul dato più recente disponibile.
